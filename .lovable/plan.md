# Futuristic Developer Portfolio for Manish

## Design System

- **Dark theme** with deep navy/black background
- **Purple-blue gradient** glow accents throughout
- **Glassmorphism** cards with backdrop-blur and semi-transparent borders
- **Neon lighting** effects using box-shadows and text-shadows
- Subtle **grid background overlay** pattern
- Clean modern typography with bold headings and good spacing

## Sections

### 1. Sticky Navbar

- Transparent background with blur effect
- Centered menu links: Home, About, Work, Contact
- Smooth scroll navigation to each section
- Subtle bottom border glow

### 2. Hero Section (Full Screen)

- Left side: Large "Hi, I'm Manish" heading, gradient-highlighted role title (e.g. "Full-Stack Developer"), short intro paragraph, rounded gradient CTA button linking to contact
- Right side: Placeholder portrait area with glowing gradient border
- Animated glowing purple/blue background orbs

### 3. About Section

- Centered "About" title with gradient text
- Professional bio paragraph
- Row of floating tech icons (React, TypeScript, Node.js, etc.) with subtle glow hover effects

### 4. Skills Visualization

- Circular/orbital layout of tech icons around a glowing center symbol
- Futuristic UI feel with ring animations and glow effects

### 5. Projects Section ("Recent Work")

- Alternating layout: image left / text right, then text left / image right
- Glassmorphism project cards with hover glow animations
- Each card: "Featured Project" label, project title, description, GitHub & live demo icon links
- 3-4 placeholder example projects

### 6. Contact Section

- Left side: "Drop me a message" heading, short description, contact info (phone, email, address) with icons
- Right side: **Working glassmorphism contact form** with Name, Email, Message fields
- Form validation using react-hook-form + zod
- Rounded gradient "Send Message" button
- Toast notification on successful submission
- full authentication with form connect to your email

### 7. Footer

- Copyright text centered
- Social media icon links (GitHub, Twitter/X, LinkedIn)

## Animations & Effects

- Scroll-triggered fade-in animations using Intersection Observer
- Hover glow effects on cards, buttons, and icons
- Smooth scrolling behavior
- Soft shadows and depth throughout

## Responsive Design

- Fully mobile responsive
- Hero stacks vertically on mobile
- Project cards stack on smaller screens
- Contact section stacks form below info on mobile
